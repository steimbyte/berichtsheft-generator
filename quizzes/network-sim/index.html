Quick Start Guide
1. Add Devices
Drag devices from the left panel onto the canvas. Devices stay draggable after placement.
2. Connect Devices
Click on a port, then drag to another device's port. Select cable type from toolbar.
3. Configure Devices
Right-click a device → Config, or use the CLI for advanced commands.
4. Test Connectivity
Click "PDU" to send test packets. Watch the animation and logs.
